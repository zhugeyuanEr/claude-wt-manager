# 完整进度报告

生成综合的多线开发进度报告。

## 使用方式

```
/wt-report
```

## 报告内容

### 1. 执行摘要
- 总线程数
- 整体进度
- 关键里程碑状态

### 2. Worktree 状态
- 各 worktree 的分支
- 工作区状态

### 3. 线程进度
- 进度条可视化 (████░░░░░ 60%)
- 检查点完成情况
- 阻塞项
- 剩余工作

### 4. 接口状态
- 已实现的接口
- 待实现的接口
- 依赖关系

### 5. 风险评估
- 已知风险
- 建议行动
