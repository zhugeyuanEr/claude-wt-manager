# 批量质量检测

对所有 worktree 进行批量质量检测。

## 使用方式

```
/wt-batch-qa
```

## 检查项

对每个 `worktree-thread-*` 目录:
1. 运行 lint 检查
2. 运行测试
3. 汇总结果

## 输出格式

```
>>> 检测: thread-ux
  ✓ Lint 通过
  ✓ 测试通过

>>> 检测: thread-api
  ✗ Lint 失败
  ✓ 测试通过

================================
结果: 2/3 通过
⚠ 1 个失败
```

全部通过输出: "✓ 全部通过"
