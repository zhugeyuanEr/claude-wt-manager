# 合并检查

Worktree 合并前的验证检查。

## 使用方式

```
/wt-merge-check thread-ux   # 检查 thread-ux 是否可以合并
```

## 检查项

1. **同步状态** - 是否已与 origin/main 或 origin/master 同步
2. **冲突检查** - 是否有未解决的冲突
3. **工作区状态** - 工作区是否干净
4. **分支信息** - 显示分支和最近提交

## 输出示例

```
>>> 同步状态检查
  ✓ 已与 origin/main 同步

>>> 冲突检查
  ✓ 无冲突

>>> 工作区状态
  ✓ 工作区干净

建议: 可以合并
```

如有问题，列出具体问题和修复建议。
